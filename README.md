
# OffEngine

An OpenGl-based simple 2D graphic engine



## Abstractions
OffEngine uses some basic abstractions, inspired by classic Scratch game "engine". 
- Costume - sigular frame of Sprites animation, defined as a set of points in relation to the sprites center x and y value
- Sprite - a rendered representation of Game Object on a screen
- Game Object - base class, after which all the objects to be rendered on the screen will inherit
